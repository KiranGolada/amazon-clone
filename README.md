# 🛒 Amazon Clone

A simple Amazon homepage clone built using **HTML** and **CSS**. This project was created to practice frontend development by recreating the layout and design of Amazon's landing page.

## 🌐 Live Demo

👉 https://kirangolada.github.io/amazon-clone/

---

## 📌 Features

- Amazon-inspired homepage layout
- Responsive navigation bar
- Search bar UI
- Hero section
- Product category boxes
- Footer similar to Amazon
- Clean and organized code structure

---

## 🛠️ Technologies Used

- HTML5
- CSS3

---

## 📁 Project Structure

```
amazon-clone/
│── index.html
│── style.css
│── images/
└── README.md
```

---

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/KiranGolada/amazon-clone.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

No additional setup is required.
